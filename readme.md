## About
Slated is a starter theme for Shopify using a mixture of elements from Slate, ThemeKit, [Shopify Skeleton](https://github.com/Pixel2HTML/shopify-skeleton) and Timber â€“ using Bulma as a CSS framework.

## Features
- ES6 support
- Sourcemaps
- Browser sync
- Slate [helper scripts](https://shopify.github.io/slate/js-examples/)
- SCSS imports and liquid variables

## Usage

```bash
# Compile and watch
$ npm run code

# zip for production
$ npm run build

# full deployment
$ npm run deploy

# delete theme files
$ npm run kill-theme
```
